## Overview
This project is a simple Digital Clock built using HTML, CSS, and JavaScript. It displays the current time in real-time and updates every second without requiring a page refresh. The main purpose of this project is to practice DOM manipulation, JavaScript timing functions, and basic frontend design skills.

## Features

Live Time Display
Shows current time in real-time.
Hours, Minutes, Seconds
Displays full time (HH:MM:SS).
Automatic Update
Time updates every second.
